AIDES DES COMMANDES

Rentrez l'action haut, droite, bas, gauche pour bouger Goco.

Rentrez l'action sauter pour gravir les obstacles.

Rentrez l'action couper pour abattre les arbres.

Rentrez l'action tirer pour éliminer les ennemis.

Rentrez dans pour vous téléporter dans un endroit.

Aller sur pour débloquer la plateforme rouge.

INSTRUCTIONS


L’instruction FOR crée une boucle composée de trois expressions optionnelles séparées par des points virgules et encadrés entre des parenthèses qui sont suivies par une instruction à exécuter dans la boucle.

L'instruction ninja.action ( ) va permettre d'entrer une action unique afin que le ninja l'éxécute.

Le bouton PLAY permet de lancer l'algorithme et de voir Goco le parcourir.

Le bouton RESET permet de recommencer depuis le début, il réinitialise le parcours ainsi que le tableau d'instructions.
